Tool for encrypting / decrypting text

How does it work: 

1. Clone the depository writing "git clone https://gitea.kood.tech/brunozambrano/cypher.git" into the terminal.

2. Go to the depository directory called cypher & write "go run ." to use the tool.

3. You can first choose to ether decrypt or encrypt your text

4. Choose one of the cypher options.

5.Input your text

6.Receive your encrypted / decrypted text

You have 3 options for encrypting / decrypting your text:

a) ROT13.
It will change each letter with another letter 13 positions after it in the alphabet. 
For example: if you give it "abc" it will return "nop".

b) Reverse.
It will return your text backwards.
For example: if you give it "hello" it will return "olleh"

c) Vowel swap.
It will change all the vowels in your text with the next vowel in the English alphabet.
For example: if you give it "abcy" it will return "ebca"
